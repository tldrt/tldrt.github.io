About TLDRT

TLDRT is a website dedicated to share quality literary fragments on Twitter and other social media platforms.

Every fragment has a unique color - a hexadecimal number that you can use to locate source URL - stamped on it.

Fragments are published as small, PDF files. The files are typeset using Markdown and LaTeX, generated using Pandoc, XeLaTeX and ImageMagick. TLDRT is hosted on Github.
